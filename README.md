## 發行時的注意事項

以下是官方文件提及的基本要求：

1. 如果要部署到 Windows 的話，僅支援 Windows 10 (或更高) 的版本
2. 目前的 Azure AI Vision SDK 僅支援 `x64` 平台
3. 目標電腦必須額外安裝 [Microsoft Visual C++ Redistributable for Visual Studio 2015, 2017, 2019, and 2022](https://learn.microsoft.com/cpp/windows/latest-supported-vc-redist)

發行程單一檔案：

```sh
dotnet publish -c Release --nologo --self-contained -r win-x64 -p:PublishSingleFile=true -p:DebugType=embedded -p:IncludeNativeLibrariesForSelfExtract=true
```
